Commandes de base



i : Passer dans le mode insertion.

A : Ajouter en fin de ligne.

o : Ouvre une ligne en insertion en dessous de la position du curseur.

O : Ouvre une ligne en insertion au dessus de la position du curseur.

:q : Quitter.

:q! : Quitter sans enregistrer.

:w : Enregistrer le fichier.

:wq : Enregistrer et quitter.

:x : Enregistrer (seulement en cas de modification) et quitter.

:v : Rentrer en mode visuel, le déplacement du curseur entraine une surlignation.
En mode visuel, taper ":" vous verrez apparaître <','>.
Taper wNOM_DE_FICHIER pour enregistrer les élements surlignés dans un fichier NOM_DE_FICHIER.

:!rm NOM_DE_FICHIER : Efface le fichier enregistré.

:set paste : Passer en mode "collage".

:! : Pour executer une commande externe.
exemple : :!ls ou :!pwd.



Commandes de déplacement



k : commande de déplacement vers le haut.

l : commande de déplacement vers la droite.

j : commande de déplacement vers le bas.

h : commande de déplacement vers la gauche.

$ : commande de déplacement jusqu'à la fin de la ligne.

e : commande de déplacement jusqu'à la fin du mot en excluant le dernier caractère.

w : commande de déplacement jusqu'au début du prochain mot.

<control>-G : Donne le numéro de la ligne.

G : commande de déplacement à la dernière ligne du fichier.

gg : commande de déplacement au début du fichier.

XG : commande de déplacement à la ligne X. X étant une variable.

Remarque : Taper un nombre avant un mouvement le repete autant de fois que le nombre l'indique.



Commandes d'édition



a : Ajouter un mot après le curseur.

u : Annuler la dernière opération.

. : Répéter la dernière opération d'édition.

yy : Copier la ligne (4yy = 4 lignes).

dd : Couper la ligne (4dd = 4 lignes).

p : Coller après (P = insérer avant).

x : Effacer le caractère.

dw : Effacer le texte jusqu'à la fin du mot.

diw : Effacer le mot sous le curseur.

d$ : coupe la phrase après le curseur.

u : annule les dernières commandes.

U : remettre la ligne à son état initiale.

p : coller une ligne ou un mot qui a été couper, sur le curseur.

rX : déplace le curseur sur une lettre à modifié, X représentant une variable.

cc : efface le mot(groupe de caractères) sous le cureur, passe en mode insertion.

<control>-r : Rétablir la dernière opération annulée.

Remarque : Taper un nombre avant le mouvement le repete autant de fois que le nombre l'indique.



Commandes de recherche/remplacement



/ : Rechercher du texte vers l'avant.

? : Rechercher du texte vers l'arrière.

n : Rechercher l'occurence suivante.

N : Rechercher l'occurence précédente.

cw : Remplacer le texte jusqu'à la fin du mot.

ciw : Remplacer le mot.

C : Remplacer jusqu'en fin de ligne.

R : Remplace plus d'un caractère.

. : Répéter la dernière opération d'édition.

:s/A/B : Remplacer A en B sur la première occurance de la ligne.

:s/A/B/g : Remplacer tous (g) les A par des B.

#,#s/A/B/g : Remplacer A par B entre deux ligne. # est une variable prenant le numéro de la ligne.

%s/A/B/g : Remplacer A par B dans tout le fichier.

%s/A/B/gc : Remplacer A par B dans tout le fichier mais demande confirmation à chaque fois.



Fenêtrage



<control-w>-n : Ouvrir un fichier vierge dans une nouvelle fenêtre

<control-w>-s : Diviser horizontalement

<control-w>-v : Diviser verticalement

<control-w>-w : Passer à la fenêtre suivante

<control-w>-q : Fermer la fenêtre



Configuration



set nocompatible : Désactive la compatibilité avec vi

set background=dark : adapte les couleurs pour un fond noir

syntax on : Active la coloration syntaxique

set number : Affiche les numéros de lignes

set tabstop=4 : Indentation toutes les quatre colonnes

set expandtab : Conversion des tabulations en espaces

set shiftwidth=4 : Indentation de quatre colonnes

set smartindent : Indentation "intelligents"

set autoindent : Conserve l'indentation courante sur les nouvelles lignes

set backspace=indent,eol,start : retours arrières intelligents

set shiftround : Indentation à la marque de Tab la plus proche

set ignorecase : Ignore la casse lors de recherche

set showmode : affiche le mode d'édition

set ruler : affiche la position du curseur

set showcmd : affiche les commandes incomplète

map <C-m> :resize : Ctrl-m maximize la fenetre courrante (:help map)

map <C-b> :bNext : Ctrl-b aller au tampon suivant (:help map)
